# Flow 01: First Launch and Browse Products

## What to Test
User opens the app for the first time and browses available products.

## Steps
1. Launch the app
2. Wait for splash screen (1.5 seconds)
3. App opens on Home tab
4. Browse featured products section
5. Check category cards (Fashion, Electronics, Books, Sports)
6. Switch to Orders tab
7. Verify empty state showing "No Orders Yet"
8. Check Profile tab shows user info

## Expected Results
- Splash screen displays and transitions smoothly
- Home tab shows 5 featured products with prices
- All 4 tabs are visible and working
- Product cards display name, price, and add button
- Empty states show proper messages
- Navigation is smooth without delays
